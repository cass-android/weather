Weather Forecast Tracker

Data from openweathermap.org for Toronto, ON

More About This Project

The goal of this project was to visualize the magintude of updates to a frequently-changing forecast. Current weather conditions and a five-day forecast are retrieved hourly, using the free OpenWeatherMap API, and stored. Forecasts are plotted and colorized, with bluer lines show predictions made further back in time.

One observation is that the forecast values all seem to lag the actual temperature changes. It is unclear whether this is an artifact, or indicative of a persistence model being used, or some other cause. Data will be archived for further analysis.
