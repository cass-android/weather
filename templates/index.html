Weather Forecast Tracker

Data from openweathermap.org for Toronto, ON

Project Description

The goal of this project was to visualize updates to a frequently-changing forecast. The redder lines show predictions made further back in time for the same time period. More detailed views can be seen by chaning the time period (top left).

Our database is updated hourly using the free OpenWeatherMap API. This gives us the current weather conditions along with a five-day forecast.

One observation is that the forecast values all seem to lag the actual temperature changes. It is unclear whether this is an artifact, or indicative of a persistence model being used, or caused by something else. Data will be archived for analysis.
